Projet

Le projet consiste à augmenter le programme de lancer de rayons développé dans le cours IFT3355 pour qu'il génère des images de volumes participatifs, intégrés dans la scène 3D courante comme une primitive régulière. A noter que deux volumes en superposition (problème mal posé) n'ont pas à être gérés correctement.

Toutes les images intéressantes qui seront générées par le programme seront publiées dans le repo Github du projet dans le dossier "Images".

Tous les rapports bi-hebdomadaires seront publiés dans le repo Github du projet dans le dossier "Rapports/{date}/{date}.tex" et "Rapports/{date}/{date}.pdf"

Objectifs

Les objectifs du projet sont les suivants:

  • Génération procédurale et lecture de volumes participatifs sous forme de grilles de valeurs dans une boîte (extension possible à OpenVDB en bonus)
  • Implémentation du 'ray marching' dans la grille (pas réguliers, pas réguliers avec jitter, milieu du rayon dans chaque voxel, milieu avec jitter)
  • Évaluation de l'atténuation le long d'un rayon (caméra, ombre, autre)
  • Évaluation de 'single scattering' avec fonction de phase (uniforme, Mie, Rayleigh, HG)

Ce qui a été accompli:

Liste des tâches accomplies!

  • Génération procédurale de la grille de voxels.
  • Implémentation du test d'intersection entre la grille et un rayon.
  • Implémementation du premier algorithme de parcours de la grille, le DDA-3D.
  • Implémentation de l'accumulation de densité le long du rayon (fonctionalité à travailler).
  • Implémentation complète de l'accumulation de densité le long du rayon.
  • Implémentation de l'interpolation trilinéaire pour obtenir la densité dans un voxel (au lieu qu'elle soit uniformément distribuée dans le voxel).
  • Implémentation du second algorithme de parcours de la grille, le Ray Marching ainsi que toutes les variantes nécessaires au projet.
  • Implémentation de la fonction de phase de de Henyey-Greenstein.
  • Écriture du rapport final du projet.
  • Génération de plusieurs images de volumes participatifs.
